# E-Commerce Web Application

A web-based e-commerce platform built as a school project to demonstrate full-stack development skills.

## 📋 Project Overview

This is a educational project developed to practice and showcase web development fundamentals including PHP backend, HTML/CSS frontend, and basic database management.

## 🛠️ Tech Stack

- **Backend:** PHP
- **Frontend:** HTML, CSS
- **Server:** XAMPP (Apache)
- **Other:** JavaScript (for interactivity)

## 📚 About This Project

This is a **school project** created for educational purposes. It demonstrates core concepts in web development and is no longer actively maintained.
